
# SOC Assignment – Advanced Security Operations Center Activities

## Overview

This repository contains the practical activities, evidence files, reports, and workflow documentation completed as part of the Security Operations Center (SOC) assignment.

The objective of this assignment was to gain hands-on experience with SOC processes including log analysis, threat intelligence integration, incident escalation, alert triage, evidence preservation, and a complete incident response workflow simulation.
