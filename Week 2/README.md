# Week 2 - SOC Operations and Incident Response

## Overview

This repository contains the Week 2 activities completed as part of SOC (Security Operations Center) training and incident response practice. The objective was to understand alert management, incident classification, incident response procedures, alert triage, evidence preservation, and the complete alert-to-response lifecycle.

---

## Learning Objectives

* Understand Alert Priority Levels
* Classify Security Incidents
* Learn the Incident Response Lifecycle
* Practice Alert Triage
* Preserve Digital Evidence
* Document Security Incidents
* Simulate a Complete SOC Investigation Workflow
