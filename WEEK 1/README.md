# SOC Analyst Learning Project

## Overview

This repository contains theoretical knowledge, practical exercises, screenshots, and supporting resources developed as part of a Security Operations Center (SOC) Analyst learning project.

The objective of this project is to understand the core responsibilities of a SOC Analyst, including security monitoring, log analysis, incident response, alert triage, and security documentation.
